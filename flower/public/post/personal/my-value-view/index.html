My Value View

我的价值观

价值观同化是人类的灾难，宗教就是把很多人的价值观念同化了，导致了格格不入的战争。也许这是人类要明白自然的道理的代价，个体该自由、独立、随心所欲的流动生活，不必和价值观不相容的人死磕，和价值观相近的人生活在一起就好了，这可能是大陆文化的通病：不自由。

恨总比爱容易放下，有时分手的说法 是对对方的一个解脱，并不一定是对方的真实想法，只不过意图很明确是分手，不再纠缠而已。

####1. 我想要的爱情：

对自己：能随心所欲的做自己想做的事，做自己认为对的事，即坚持自己的价值观，有一份做起来有热情、不虚度光阴的工作，开心、简单，没有很多利益人情牵绊，不为别人操纵控制，有尊严的活着。

对爱人：独立、成熟、可爱、深情、专一、浪漫、有人性的善良正义之美（格局大）。不为世俗的条条框框所束缚，自然、勇敢。看的顺眼，不需要我担心背叛，天涯海角，心里也只有对方，不作他想。不功利，不挑剔嫌弃，有同理心和爱心。

对关系：自由、平等，凡事可商量、讲道理，彼此努力靠近，寻找双方最舒服的相处方式。

人世间唯真爱和自由不可辜负，如果有后来他们会幸福吗？也许会也许不会，没有如果。当越来越多的人说追求诗和远方的人是疯子，我倒觉得他们是勇敢的自由骑士，疯了的，是这个世界。金钱的分量被过于夸大了。
